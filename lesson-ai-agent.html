AI Agent
是什么？

从"会聊天"到"会做事"—— 智能体，是能自己规划、调用工具、完成任务的 AI。

本质区别

大模型回答问题，Agent 替你办事

普通大模型像一个"无所不知的顾问"，你问它答；而 Agent 更像一个"能干活的助理"—— 你给它一个目标，它会自己拆解步骤、调用工具、一步步执行，直到把事情办成。

它怎么干活

感知 → 规划 → 调用工具 → 执行

1

理解目标

听懂你要什么，比如"帮我订一张明天去上海最便宜的高铁票"。

2

拆解规划

把大目标拆成小步骤：查车次 → 比价 → 选座 → 下单 → 付款。

3

调用工具

调用搜索、订票接口、支付等工具来真正执行每一步，而不只是"说说"。

4

反馈与纠错

遇到问题会自己调整方案、重试，最终把结果交付给你。

能用来做什么

从写代码到跑流程，它都能上

编程

自动写代码、改 Bug

读懂需求后自己改文件、跑测试、修复错误。

办公

自动整理资料、出报告

检索、汇总、生成文档，一条龙完成。

运营

自动跑流程、发内容

按设定的工作流定时执行重复性任务。
